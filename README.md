# Agenda Psicólogo

This is a code bundle for the Agenda project. The original project is available at https://www.figma.com/design/R9cqTIKIzYxKXVuEzalgsq/Sistema-de-Reservas-Psic%C3%B3logo.

## Running the code

Run `npm i` to install dependencies.

Run `npm run dev` to start the development server.

## Supabase

1. Copy `.env.example` to `.env` and set your keys.
2. Follow the full setup in `SUPABASE-SETUP.md`.

If Supabase env vars are not configured, the app runs with local mock data.
